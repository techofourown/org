## Boundary note

The public policy documents in this repository record **posture and requirements**.

Detailed secret inventories, continuity-bundle composition, device configuration
profiles, storage arrangements, drill checklists, and break-glass procedures are
maintained in TOOO-controlled private operations documentation rather than in
this public repository.
